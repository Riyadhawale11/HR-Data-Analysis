HR Data Analysis Using Python & Jupyter Notebook

This project contains an exploratory analysis of HR employee data, performed using Python in a Jupyter Notebook.
The goal of the analysis is to identify key HR insights such as employee distribution, performance patterns, salary trends, attrition-related factors, and workforce structure.

 Key Highlights of the Analysis

Cleaned and prepared HR dataset for analysis

Handled missing values and data inconsistencies

Performed exploratory data analysis (EDA)

Visualized key HR metrics and employee trends

Identified patterns that can help HR teams make data-driven decisions
